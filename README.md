# ez_mouse
## 模仿自[OneQuick](https://github.com/XUJINKAI/OneQuick.AHK-legacy)

通过鼠标快速操作音量，多媒体，翻页

鼠标移动到屏幕左上角
  滚轮向上 / 前侧键 : 音量提升
  滚轮向下 / 后侧键 : 音量降低
  滚轮按下 : 静音

鼠标移动到屏幕左下角
  滚轮向上 / 前侧键 : 上一曲
  滚轮向下 / 后侧键 : 下一曲
  滚轮按下 : 播放 / 暂停
  
鼠标移动到屏幕左下角
  滚轮向上 : 向上翻页
  滚轮向下 : 向下翻页
  滚轮按下 : 禁用 / 启用 翻页功能
  前侧键 : 禁用 / 启用 音量调节
  后侧键 : 禁用 / 启用 多媒体控制

程序启动后会在任务栏托盘里生成一个图标，右键也可调节功能是否启用，图标的三条横杠由上而下分别是 [音量调节, 多媒体控制, 翻页] 功能启用状态，绿色代表启用，红色代表禁用
